# Product Requirements Document (PRD)

## Sistem Klasifikasi untuk Prediksi Kelas Dataset

---

## Ringkasan

Proyek ini bertujuan mengembangkan model klasifikasi machine learning untuk memprediksi kelas kategori dalam dataset tertentu, seperti dataset Iris, dengan menggunakan minimal dua algoritma klasifikasi. Model akan dievaluasi untuk membandingkan performa dan kesimpulan dibuat berdasarkan hasil evaluasi tersebut.

---

## Tujuan

- Menghasilkan model klasifikasi yang efektif dan akurat dalam memprediksi kelas.
- Melakukan eksplorasi data dan preprocessing yang tepat untuk persiapan data.
- Membandingkan performa minimal dua algoritma klasifikasi pada dataset.
- Menyusun laporan singkat yang informatif mengenai proses dan hasil.

---

## Lingkup Pekerjaan

- Pemilihan dataset dengan target kategorikal (misal: Iris).
- Eksplorasi data dan preprocessing meliputi visualisasi, penanganan missing value, dan transformasi fitur.
- Implementasi dan pelatihan model klasifikasi minimal dua algoritma (Logistic Regression, Decision Tree, dll).
- Evaluasi model menggunakan confusion matrix, accuracy, precision, recall, F1-score, dan ROC Curve.
- Dokumentasi kode dalam notebook atau script, serta laporan singkat hasil pengerjaan.

---

## Kriteria Keberhasilan

- Akurasi model minimal 85% untuk prediksi kelas.
- Dokumentasi kode yang jelas dan mudah diikuti.
- Laporan singkat yang ringkas dan menyajikan hasil evaluasi dengan baik.

---

## Deliverables

- File notebook (.ipynb) atau script Python (.py) berisi seluruh pipeline pengerjaan.
- Laporan singkat maksimal 3 halaman berisi deskripsi dataset, algoritma yang digunakan, evaluasi, dan kesimpulan.

---
